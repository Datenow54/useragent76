GitHub Pages সেটআপ গাইড

বিনামূল্যে হোস্টিং - শুধু HTML, CSS, JavaScript ব্যবহার করে ওয়েবসাইট তৈরি করুন 🚀

✨ GitHub Pages কি?

GitHub Pages হল একটি ফ্রি হোস্টিং সার্ভিস যা আপনার GitHub রিপোজিটরি থেকে সরাসরি স্ট্যাটিক ওয়েবসাইট পাবলিশ করে। পোর্টফোলিও, প্রজেক্ট ডকুমেন্টেশন বা পার্সোনাল ব্লগের জন্য পারফেক্ট! [citation:6][citation:8]

⚠️ আগে জানা জরুরী

পাবলিক রিপোজিটরি: ফ্রি প্ল্যানে রিপোজিটরি পাবলিক থাকতে হবে।
স্ট্যাটিক সাইট: শুধু HTML, CSS, JS সাপোর্ট করে (PHP বা ডাটাবেস থাকবে না) [citation:8]।
ইনডেক্স ফাইল: হোমপেজের নাম index.html রাখতে হবে [citation:3][citation:6]।

১

GitHub একাউন্ট ও রিপোজিটরি তৈরি

প্রথমে একটি GitHub একাউন্ট খুলুন (যদি না থাকে)। এরপর:

  • লগইন করে ডান পাশের "+" আইকনে ক্লিক করুন
  • "New repository" সিলেক্ট করুন [citation:6]
  • Repository Name: আপনার-ইউজারনাম.github.io (উদাহরণ: sakib.github.io)
  • Public সিলেক্ট করে "Create repository" ক্লিক করুন [citation:8]
💡 টিপ: বিশেষ রিপোজিটরি নাম username.github.io ব্যবহার করলে সাইটটি সরাসরি https://username.github.io এ ওপেন হবে [citation:3][citation:9]!
২

HTML ফাইল আপলোড করা

রিপোজিটরিতে আপনার ওয়েবসাইটের ফাইল যোগ করুন।最简单的方法是:

  • রিপোজিটরির ভিতরে "Add file" → "Upload files" ক্লিক করুন
  • আপনার তৈরি করা index.html ফাইল ড্র্যাগ করুন [citation:8]
  • নিচে "Commit changes" বাটনে ক্লিক করুন
<!-- একটি মিনিমাম HTML উদাহরণ -->
<!DOCTYPE html>
<html>
<head>
    <title>মাই ফার্স্ট সাইট</title>
</head>
<body>
    <h1>হ্যালো ওয়ার্ল্ড!</h1>
    <p>এটা আমার GitHub Pages সাইট।</p>
</body>
</html>
৩

GitHub Pages অ্যাক্টিভেট করা

রিপোজিটরির সেটিংস থেকে GitHub Pages চালু করতে হবে:

  • রিপোজিটরির Settings ট্যাবে যান [citation:5][citation:7]
  • বাম সাইডবার থেকে Pages মেনুতে ক্লিক করুন [citation:3]
  • Branch অপশনে main সিলেক্ট করুন
  • Folder এ /(root) সিলেক্ট করে Save ক্লিক করুন [citation:3]
⏱️ সময়: সেভ করার পর ১-২ মিনিট অপেক্ষা করুন। রিপোজিটরি রিলোড দিলে উপরে একটি নোটিশ দেখাবে: "Your site is published at https://username.github.io" [citation:6]
৪

সাইট ভিজিট করা ও আপডেট করা

এবার আপনার ব্রাউজারে https://username.github.io ওপেন করুন (username আপনার ইউজারনেম দিয়ে পরিবর্তন করে)।

আপডেট প্রসেস: নতুন HTML/CSS ফাইল আপলোড করলেও GitHub Pages স্বয়ংক্রিয়ভাবে আপডেট হবে। মাঝে মাঝে ২-৩ মিনিট লেগে যেতে পারে [citation:1][citation:8]।

🔄 বারবার আপডেটের নিয়ম: লোকাল ফাইল এডিট করে git commands ব্যবহার করতে চাইলে: 
git add --all
git commit -m "আপডেটের কারণ লিখুন"
git push
উপরের কমান্ডগুলো আপনার কম্পিউটারে Git ইন্সটল করা থাকবে। না থাকলে সরাসরি GitHub ওয়েবসাইট থেকে ফাইল আপলোড দিতে পারেন [citation:6][citation:8]।

🎨 থিম ও কাস্টম ডোমেইন (অপশনাল)

থিম সিলেক্ট: Settings → Pages → Theme Chooser থেকে বিল্ট-ইন থিম সিলেক্ট করে দিতে পারেন [citation:9]।
কাস্টম ডোমেইন: আপনার নিজের ডোমেইন (যেমন myname.com) কানেক্ট করতে চাইলে, সেটা সম্ভব। এর জন্য রেজিস্ট্রার থেকে ডোমেইন কিনতে হবে [citation:1][citation:2]।
